# y2mate Youtube downloader

- Program využívá stránku [y2mate.com](https://www.y2mate.com/)
- Program byl vytvořen v [QT creatoru](https://www.qt.io/product/development-tools)
- Verze knihovny: QT6
- Primárně stavěno pro windows 64bit
